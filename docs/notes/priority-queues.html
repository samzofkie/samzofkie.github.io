Priority Queues

A priority queue is a data structure for maintaining a set of key-value pairs ordered (max or min) by the value, supporting the operations:

  • INSERT(S, k, v): inserts k, v into set S
  • MAXIMUM(S): returns pair from S with the largest value
  • EXTRACT-MAX(S): returns and removes pair from S with the largest value
  • INCREASE-VALUE(S, k, v): increases k's value to v, which is assumed to be at least as big as k's old value

Heaps operate on numeric values stored in an array, so a schema must be worked out to map numeric values in a heap to meaningful objects in application code. An auxiliary hash table, for instance, can do achieve this adding only O(1) time overhead.
